# Projeto em React

### Passos para rodar o servidor

1. Crie um banco de dados chamado 'sparkenergydb' no mysql
   1.2. Altere as outras configurações caso necessário
2. Configure o banco de dado no arquivo './config/config.json'
3. Use no terminal dentro da pasta 'npm start'
4. Pronto, servidor iniciado e conectado com o bando de dados
